# Amazon Sales Analysis Dashboard

## Overview

An interactive Power BI dashboard developed to analyze Amazon Fashion sales data and provide actionable business insights through dynamic reporting and KPI visualization.

The dashboard enables users to monitor sales performance, analyze product categories, identify regional trends, and explore product-level information using interactive filters and drill-through pages.

---

## Features

- Interactive dashboard with multiple report pages
- Dynamic KPI cards
- Product-wise sales analysis
- Category-wise and State-wise performance
- Time-series sales and profit trends
- Product catalog with images
- Drill-through product details
- Search functionality
- Dynamic slicers and filters

---

## Dashboard Pages

### 1. Overview

Provides a high-level business summary including:

- Total Sales
- Total Units Sold
- Seller Count
- Sales by Category
- Sales by State
- Sales Trend Over Time

---

### 2. Insights

Focused on profitability analysis.

Includes:

- Profit by Category
- Profit by State
- Profit Trend
- Dynamic KPI Cards
- Interactive filtering

---

### 3. Products

Displays the complete product catalog with images.

Users can:

- Browse products
- Filter by category
- Search products
- Navigate to detailed product information

---

### 4. Product View

Dedicated product analysis page showing:

- Product Image
- Sales Amount
- Units Sold
- Average Rating
- Returns/Loss
- Time-based Sales Trend
